# Palindrome Checker App

## Objective

The objective of the **Palindrome Checker App** is to design and implement a console-based Java application that validates whether a given string is a palindrome.
This project helps in understanding basic Java programming concepts and string manipulation.

---

## Use Case 3 – Palindrome Check Using String Reverse

### Goal

Check whether a string is a palindrome by reversing the string and comparing it with the original string.

### Flow

1. The program starts.
2. A string is defined in the program.
3. The string is reversed using a **for loop**.
4. The reversed string is compared with the original string.
5. If both strings are equal, it is a **Palindrome**.
6. The result is printed on the console.

### Key Concepts Used

* **Class** – Container for the application logic.
* **Main Method** – Entry point of the Java program.
* **String** – Used to store the text that will be checked.
* **For Loop** – Used to reverse the string.
* **String Concatenation (+)** – Used to build the reversed string.
* **equals() Method** – Used to compare the original and reversed string.

### Example

Input:

```
madam
```

Output:

```
madam is a Palindrome
```

### Program Compilation and Execution

Compile the program:

```
javac UseCase3PalindromeCheckerApp.java
```

Run the program:

```
java UseCase3PalindromeCheckerApp
```
